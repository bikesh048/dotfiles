
> Each directory inside `dotfiles/` is a stow package.

# macOS
brew install stow

# Ubuntu/Debian
sudo apt install stow

### Stow Configuration
stow zsh
stow nvim
stow zellij
stow wezterm


stow zsh     # creates ~/.zshrc
stow nvim    # creates ~/.config/nvim
stow zellij  # creates ~/.config/zellij
stow wezterm # creates ~/.config/wezterm
